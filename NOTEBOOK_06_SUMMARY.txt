COMPREHENSIVE EVALUATION FRAMEWORK (4 Levels):
================================================================================

LEVEL A: Individual Fold Predictions (No TTA)
  - Each of 5 fold models predicts on test set independently
  - No augmentation applied
  - Baseline for ensemble performance

LEVEL B: 5-Fold Ensemble (No TTA)
  - Average probabilities from 5 fold models
  - No augmentation
  - Soft ensemble baseline

LEVEL C: 5-Fold Ensemble + 8× TTA
  - Each fold uses 8× Test-Time Augmentation
  - Applies: horizontal flip, vertical flip, random 90° rotations
  - Average across all TTA passes and all folds
  - State-of-art ensemble approach

LEVEL D: 5-Fold Ensemble + TTA + Optimized Threshold
  - Same as Level C
  - Threshold optimized using Youden's J metric
  - Clinically-relevant operating point

- Output: 95% confidence intervals for all ensemble methods
